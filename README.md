## wrt-cli

npm i wrt-cli -g
全局安装脚手架

## wrt list

列出可用的列表

## wrt -v

版本查询

## wrt init <templateName>

初始化模板

### npm link

即可把 scion 命令绑定到全局，以后就可以直接以 scion 作为命令开头而无需敲入长长的 node scion 之类的命令了

## npm publish

切回源
npm config set registry https://registry.npmjs.org/

国内源
npm config set registry https://registry.npm.taobao.org/
